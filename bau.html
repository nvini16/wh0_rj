• Fontes para títulos e seções principais (modernas e legíveis)

    • Poppins-> moderna, limpa e muitas usadas em interfaces.
    • Montserrat-> òtima para títulos e seções chamativas.
    • Raleway-> elegante, ideal para subtítulos.
    • Oswald-> boa para títulos fortes e técnicos.



  • Fontes para textos corridos (leitura confortável)

    • Roboto-> versátil, ótima para leitura em telas.
    • Open Sans-> limpa e fácil de ler em parágrafos longos.
    • Lato-> moderna e confortável para textos corridos.
    • Muli-> simples e agradável para leitura contínua.



  • Fontes monoespaçadas (para códigos e exemplosm práticos)

    • Fire Code> inclui ligaduras para símbulos de programação.
    • jetBrains Mono> criada especifivamente para desenvolvedores.
    • Consolas> Classica e legível em qualquer sistema.
    • Courier New> tradicional, compatível iniversalmente.



  • ombinações recomendadas

    • Poppins (títulos) + Roboto (texto) + Fira Code (código)
    • Montserrat (títulos) + Lato (texto) + jetBrains Mono (código)
    • Raleway (títulos) + Open Sans (texto) + Consolas (código)
