HTML — A Estrutura da Web

O que é HTML?..

HTML (HiperText Markup Language, ou Linguagem de Marcação de Hipertexto) é a base strutural de toda a página da web.
Ele não é uma linguagem de programação, mas sim uma linguagem de Marcação, usada para organizar, hierarquizar e descrever o conteúdo que será exibido no navegador.

Tudo o que você vê em um site — textos, títulos, imagens, links, botões — nasce do HTML.

Ele é o esqueleto da página, que depois será estilizado com CSS e tornado dinâmico com JavaScript.

Como ele funciona

O HTML organiza o conteúdo em elementos, chamados tags.
Cada elemento tem uma função específica e uma sintaxe própria:

<p>HTML:<br><br> <tag>atributo="valor"conteúdo</tag></p>

Saída:"valor"

<p>Por exemplo:<br><br><p>Este é um parágrafo de texto.</p></p>

Saída:Este é um parágrafo texto.

O navegador lê essas marcações e entende compo deve mostrar o conteúdo ao usuário

Estrutura básica de um docuento HTML

Todo arquivo HTML segue uma estrutura padrão, que indica ao navegador como interpretar o conteúdo:

<!DOCTYPE html> <html lang="pt-BR">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Meu Primeiro Site</title>
</head>

<body>
<h1>Olá, mundo!</h1>
<p>Este é o meu primeiro parágrafo em HTML.</p>
</body>
</html>

Explicando por parte:

  • <!DOCTYPE html> -> informa ao navegador que o documento usa HTML5.
  • <html> -> elemento raiz da página
  • <head> -> contém metadados (informações sobre o site, e não o conteúdo em si).
  • <meta charset="UTF-8"> -> define o conjunto de caracteres (permite acentos e cedilhas).
  • <title> -> título exibido na aba do navegador.
  • <body> -> contém todo o conteúdo visível de uma página.

Conceito fundamental: hierarquia

O HTML é hierarquico, ou seja, um elemento pode conter outros dentro dele.
Isso cria blocos lógicos que o navegador e o CSS conseguem manipular facilmente.

Exmplo:

<section>
<h2>Sobre mim</h2>
<p>Sou um progrador em aprendizado constante.</p>
</section>

Elementos

Um elemnto HTML é a unidade fundamental de uma página web.
Ele representa um comportamento da estrutura (como um título, parágrafo, link, imagem, botão...) e é interpretado pelo navegador para exibir o conteúdo de forma adequada.

Estrutura básica de um elemnto

Um elemnto HTML é composto, geralmente, por três partes:

<tag-de-abertura>conteúdo</tag-de-fechamento>

EX.:

<p>Este é um parágrafo.</p>

  • <p>:Tag de abertura (indica o início de elemnto).
  • </p>:Tag de fechamento (indica o fim do elemnto).
  • este é um parágrafo.:Conteúdo exibido na página.

Elentos com e sem conteúdo

  1. Elemntos com conteúdo (com tag de fechamneto)
    São aqueles que possuem um conteúdo entre as tags.

    EX.:

    <h1>Título principal</h1>
    <p>Texto do parágrafo.</p>
    <a href="#">Link</a>

  2. Elemntos vazios (sem tag de fechamneto)
    também chamados de elemntos autoencerráveis, não possuem conteúdo inteiro.

    EX.:

    <img src="foto.jpg" alt="Foto de exemplo">
    <br>
    <hr>
    <input type="text">

    Observações:
    em HTML5, não é obrigatorio usar a barra final (/>), mas ela é aceita por compatibilidade com XHTML, por exemplo:

    <img src="foto.jpg" alt="foto" />

Elentos aninhados (Nesting)

Um elemnto pode conter outros elemntos dentro dele, criando hierarquias.

EX.:

<div>
<h2>Título interno</h2>
<p>Texto dentro da visão.</p>
</div>

Aqui:

  • <div>:é o elemnto pai.
  • <h2> e <p>são filhos de <div>
Importante:

Sempre feche as tags na ordem inversa à que foram abertas, para evitar erros de estrutura.

Tipos de elemntos HTML

Os elementos podem ser classificados segundo seu comportamneto visual e estrutural:

Tipo Descrição Exemplos
Bloco (block-level) Ocupe toda a largura disponível e começa em uma nova linha. <div>, <p>, <h1>-<h6>, <ul>, <section>
inline Ocupe apenas o espaço necessário ao seu conteúdo, sem quebrar linha. <span>, <a>, <strong>, <em>, <img>
inline-block Compota-se como inline, mas permite definir largura e altura. <button>, <input>, 7ltselect>

Semântica de elemntos

A partir do HTML5, os elemntos passaram a ter funções semânticas, ou seja, informam o propósito do conteúdo para o navegador e macanismo de busca.

EX.:

<headre>Topo da página</header>
<main>Conteúdo principal</main>
<footer>Rodapé</footer>

Esses elemntos substituem o uso genérico de <div> e ajudam na organização e acessibilidade da página.

Tipos principais de elemnetos
Tipo Exemplo função
Estruturais <header>, <main>, <footer> Oragnizam o layout da página
Textuais <h1>, <p>, <strong>, <em> Criam e formatão textos
Mídia <img>, <video>, <audio> Inserem imagens, videos e sons
Navegação <a>, <nav> Criam links e menus
Formulários <form>, <input>, <label>, <button> Capturam dados do usuário

Atributos

Os atributos são informações adicionais dentro de uma tag HTML.
Eles modoficam o comportamento ou definem propriedades de um elemento. São sempre escritos dentro de uma tag de abertura.

A sintaxe geral é:

<tag atributo="valor">conteúdo</tag>

Por exemplo:

<img src="foto.jpg" alt="Descrição da foto">

Um atributo é composto por duas partes:

  1. Nome do atributo> indica o tipo de informasções (ex.: src, href, class, id).

  2. Valor do atributo> define o conteúdo ou a configuração desse atributo (ex.:"foto.jpg", "texto", "prncipal").

Aqui:

  • src > atributo que indica o caminho da imagem.
  • alt > Atributo que fornece um texto alternativo, caso a imagem não carregue.

Exemplo prático

<img src="foto.jpg" alt="Descrição da foto">

Aqui, o elemento <img> usa dois atributos:

src="foto.jpg" indica onde a imagem está armazenada.

alt="Descrição da foto" fornece um texto alternativo, caso a imagem não carregue.

Tipos de atributos

Podem ser usados em qualquer tag.

EX.:

  • id:indentifica de forma única um elemnto na página.
  • class:agrupa elemntos que compartilham o memso estilo ou comportamneto.
  • style:aplica estilos CSS diretamnete no elemnto.
  • title:adiciona uma dica (tooltip) exibida ao passar o mouse.

<p id="descricao" class="texto" title="texto informativo">Conteúdo do parágrafo</p>

Atributos específicos

São usados apenas em determinadas tags e definem comportamentos próprios.

EX.:

  • href(em <a>): define o link de destino.
  • src(em <img>): define o caminho de um arquivo externo.
  • type(em <input> ou <button>): especifica o tipo do elemnto.
  • value(em <input>): define o valor inicial do campo.

<a href="https://www.exemplo.com">Visite o site</a>
<input type="text" value="Digite aqui">

Boas práticas ao usar atributos

  • Sempre coloque os valores entre aspas duplas (""), mesmo que não sejam obrigatórias.
  • Evite usar o mesmo id em mais um elemnto.
  • Use class para aplicar estilos em grupo e id apenas quando precisar indentificar algo único.
  • Prefira atributos semânticos e padronizados, evitando nomes inventados (a não ser em data attributes).

Atributos personalizados (data-attributes)

O HTML5 introduziu os atributos personalizados que começam com data-.
Eles permitem armazenar informações extras em elemntos, acessíveis via javaScript.

EX.:

<div data-user="nome" data-role="admin">informações do usuário</div>

Esses dados podem ser manipulados em javaScript:

let user = document.querySelector('div').dataset.user; // "nome"

Tipos de atributos

Podem ser usados em qualquer elemnto HTML.
Exemplos principais:

Atributo função Exemplo
id Identificador único (usado em CSS e JS) <p id="intro">...</p>
class Agrupa elemntos semelhantes <div class="caixa"></div>
title Texto exibido ao passar o mouse <button title="clique aqui">ok</button>
style Adiciona estilos diretos (CSS inline) <p style="color: red;">Texto</p>
hidden Oculta o elemnto <p hidden>Você não é isso</p>
tabindex Define a ordem de navegação pelo teclado <input tabindex="1">

Complementação

Além dos elementos e atributos básicos, o HTML oferece recursos adicionais para enriquecer a estrutura e funcionalidade das páginas web.

Comentários

Os comentários são trechos de texto ignorados pelo navegador.
Eles servem para documentar o código, explicar partes complexas ou deixar notas para outros desenvolvedores.

EX.:

<!-- Este é um comentário em HTML -->

Entidades HTML

As entidades HTML são códigos especiais usados para representar caracteres reservados ou símbolos que não podem ser inseridos diretamente no código.

EX.:

&lt; representa o símbolo <br>
&gt; representa o símbolo >br>
&amp; representa o símbolo &br>
&quot; representa aspas duplas "br>
&#39; representa aspas simples 'br>

Doctype

A declaração doctype informa ao navegador qual versão do HTML está sendo usada.
Em HTML5, a sintaxe é simples:

EX.:

<!DOCTYPE html>
